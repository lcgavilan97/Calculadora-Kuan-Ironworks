# Kuang Ironworks — Calculadora de Herreria

Calculadora web para presupuestos de herreria con estilo western. Permite calcular totales minimos y maximos a partir de una lista de materiales con precios editables.

## Caracteristicas

- Lista de 28 materiales de herreria
- Precios minimo y maximo editables (acotados segun valores base)
- Calculo automatico de totales por material y generales
- Diseño western inspirado en Red Dead Redemption
- Responsive (funciona en moviles y desktop)
- Sin dependencias ni servidor — se abre directo en el navegador

## Stack

- HTML5 + CSS3 + JavaScript (vanilla)
- Tipografias: Playfair Display + Special Elite (Google Fonts)
- Desplegado en Vercel

## Deploy

```bash
vercel --prod
```
